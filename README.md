# SecAI - AI Secretary

SecAI is an AI-powered daily planning assistant designed to help users organize their tasks in a calmer and more natural way. Instead of manually creating a schedule from scratch, users can simply tell SecAI what they need to do, and the app helps turn those thoughts into a clear timeline for the day.

## About the Project

SecAI was built to act like a personal AI secretary. It helps users process their tasks, appointments, errands, and reminders by turning a messy “brain dump” into an organized daily plan.

The goal of the project is to make planning feel less overwhelming. Users do not need to format their tasks perfectly. They can type naturally, upload an image, or describe what is on their plate, and SecAI will help structure everything into a schedule.

## How It Works

1. The user creates an account or logs in.
2. The user opens the Huddle page and tells SecAI what they need to do.
3. SecAI reads the user’s message and identifies tasks, priorities, estimated durations, and fixed times.
4. The user can confirm or adjust the extracted tasks.
5. SecAI generates a timeline-style schedule.
6. The user can view their organized day on the Timeline page.
7. If plans change, the user can add, remove, or re-shuffle tasks.

## Main Features

- AI-powered task planning
- Natural language task input
- Screenshot or image upload for task extraction
- Automatic schedule generation
- Timeline view for daily plans
- Task prioritization
- Re-shuffle option when plans change
- User accounts and profiles
- Clean and calm interface design

## Purpose

SecAI is made for people who feel overwhelmed by scattered tasks and want a softer, easier way to plan their day. It is especially useful for students, professionals, and anyone who prefers talking through their tasks instead of filling out rigid planning forms.
